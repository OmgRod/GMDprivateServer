## GMDprivateServer
### A server emulator for Geometry Dash. Supports GD from 1.0 to 2.207

## Prerequisites

- Web server (Apache/Nginx recommended) or Docker (for containerized deployment)
- PHP 7.0 or higher
- MySQL/MariaDB database
